# Tech Portfolio & Blog

개발자를 위한 기술 블로그와 포트폴리오가 결합된 개인 웹사이트입니다.

## 🚀 주요 기능

- **포트폴리오**: 프로젝트 소개 및 기술 스택
- **기술 블로그**: 개발 경험과 지식 공유
- **뉴스레터**: 월간 기술 동향 및 인사이트
- **반응형 디자인**: 모든 디바이스에서 최적화된 경험

## 🛠 기술 스택

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Deployment**: GitHub Pages
- **CI/CD**: GitHub Actions

## 🎨 디자인 특징

- 미니멀하고 모던한 디자인
- 한국어 최적화
- 접근성 고려
- 빠른 로딩 속도

## 📝 개발 가이드

### 로컬 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 미리보기
npm run preview
```

### 프로젝트 구조

```
src/
├── components/     # React 컴포넌트
├── data/          # 정적 데이터
├── types/         # TypeScript 타입 정의
└── App.tsx        # 메인 앱 컴포넌트
```

## 🚀 배포

GitHub Pages를 통해 자동 배포됩니다:

1. `main` 브랜치에 push
2. GitHub Actions 워크플로우 실행
3. 자동으로 사이트 업데이트
